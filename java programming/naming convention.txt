Java Naming conventions:
------------------------
-->The name must not contain any white spaces and the name should not start with special characters 

CLASSES:
--------
-->It should start with the uppercase letter.
-->It should be a noun.eg(Employee,Loan,Student,etc)
INTERFACE:
----------
-->It should start with the uppercase letter.
-->It should be an adjective.eg(Bank,Loan...etc)
-->Use proper words.
METHOD:
-------
-->It should start with lowercase letter.
-->It should be a verb such as main(), print(), println(),display()...
-->If the name contains multiple words, start it with a lowercase letter followed by an uppercase.
   eg:rateOfInterest.
VARIABLE:
---------
-->It should start with a lowercase letter such as id, name.
-->It should not start with the special characters.eg(_)
-->If the name contains multiple words, start it with the lowercase letter followed by an uppercase letter.
PACKAGE:
--------
-->It should be a lowercase letter such as java, lang,bank.
-->If the name contains multiple words, it should be separated by dots (.) such as java.util, java.lang.
CONSTANT:
---------
-->It should be in uppercase letters.
-->If the name contains multiple words, it should be separated by an underscore(_).eg(AVG_MARK)
-->It may contain digits but not as the first letter.
